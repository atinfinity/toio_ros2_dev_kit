# toio_description

## Introduction

`toio_description` provides a 3D model of the [toio](https://toio.io/) for visualization.

![](https://raw.githubusercontent.com/atinfinity/toio_description/refs/heads/main/image/toio_description.png)

- ROS 2 Jazzy
