toio_gazebo

Introduction

This is a ROS 2 Package to develop package of toio using Gazebo.

Feature

toio_gazebo provides Gazebo environment for toio.

  • Control using the /cmd_vel topic
  • Publish /tf topic(position and orientation of toio)

Requirements

  • Ubuntu 24.04
  • ROS 2 Jazzy
  • Gazebo Harmonic
